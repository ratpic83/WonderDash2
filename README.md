# WonderDash2 – Motorrad-Dashboard-App (DMD2-Alternative)

Native Android (Java, API 19) app for motorcycle dashboard & navigation, optimized for offline use, OBD-II data, and modular extensibility.

## Features (Initial Skeleton)
- Modular structure for navigation, OBD-II, dashboard, music, weather, GPX, and data management
- Bottom navigation bar for quick access to Dashboard, Map, Music, and Settings
- Day/Night theme switching
- Placeholder fragments for each main section

## Requirements
- Android 4.4.4 (API 19, 32-bit ARM)
- No Google Play Services required
- Open-source libraries: OSMDroid, MapsForge, OkHttp, Gson, AppCompat v7

## Getting Started
1. Open in Android Studio
2. Build APK for API 19 (minSdkVersion 19)
3. Sideload APK to target device
